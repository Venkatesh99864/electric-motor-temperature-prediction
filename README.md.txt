Electric Motor Temperature Predictor

This is a machine learning web app that predicts the **PM (Permanent Magnet) temperature** of an electric motor.


Input Features:
- Ambient
- Coolant
- U_d
- U_q
- Motor Speed
- I_d
- I_q

ML Models Used
- Linear Regression
- Decision Tree
- Random Forest
- Support Vector Machine
